Comment créer un compte Aurora ?

Pour créer un compte Aurora :

  1. Cliquez sur le bouton "Connexion" en haut à droite
  2. Sélectionnez "Créer un compte"
  3. Remplissez le formulaire avec vos informations (nom, email, mot de passe)
  4. Acceptez les conditions d'utilisation
  5. Vérifiez votre email pour activer votre compte

Support Technique

Configuration système requise

Configuration minimale :

  • Navigateur : Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  • Connexion Internet : 5 Mbps minimum
  • RAM : 4 Go minimum
  • Processeur : Dual-core 2.0 GHz

Configuration recommandée :

  • Connexion Internet : 10 Mbps ou plus
  • RAM : 8 Go ou plus
  • Processeur : Quad-core 2.5 GHz ou plus
  • Carte graphique dédiée pour l'univers 3D

Horaires du Support

  • Email : 24/7 (réponse sous 24h)
  • Chat : Lun-Ven 9h-18h (CET)
  • Support Premium : 24/7 pour les abonnés Astronaute
