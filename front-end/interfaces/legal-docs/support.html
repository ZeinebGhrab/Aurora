Questions Fréquentes

Démarrage

Comment créer un compte Aurora ?

Pour créer un compte Aurora :

  1. Cliquez sur le bouton "Connexion" en haut à droite
  2. Sélectionnez "Créer un compte"
  3. Remplissez le formulaire avec vos informations (nom, email, mot de passe)
  4. Acceptez les conditions d'utilisation
  5. Vérifiez votre email pour activer votre compte

Note : Vous devez avoir au moins 13 ans pour créer un compte.

Quels sont les différents niveaux de difficulté ?

Aurora propose 5 niveaux de difficulté :

  • Débutant : Idéal pour découvrir un nouveau domaine
  • Intermédiaire : Pour approfondir vos connaissances
  • Avancé : Défis complexes nécessitant de l'expérience
  • Expert : Pour les utilisateurs très expérimentés
  • Maître : Les défis les plus difficiles de la plateforme

Le système s'adapte automatiquement à votre progression.

Mon Compte

Comment modifier mes informations personnelles ?

Pour modifier vos informations :

  1. Connectez-vous à votre compte
  2. Cliquez sur votre avatar en haut à droite
  3. Sélectionnez "Paramètres du compte"
  4. Modifiez les informations souhaitées
  5. Cliquez sur "Enregistrer les modifications"

J'ai oublié mon mot de passe, que faire ?

Pour réinitialiser votre mot de passe :

  1. Cliquez sur "Connexion"
  2. Sélectionnez "Mot de passe oublié ?"
  3. Entrez votre adresse email
  4. Consultez votre boîte mail (vérifiez les spams)
  5. Cliquez sur le lien de réinitialisation
  6. Créez un nouveau mot de passe

Important : Le lien expire après 24 heures.

Comment supprimer mon compte ?

Pour supprimer votre compte :

  1. Allez dans "Paramètres du compte"
  2. Descendez jusqu'à "Supprimer mon compte"
  3. Lisez les informations sur la suppression
  4. Cliquez sur "Supprimer définitivement"
  5. Confirmez avec votre mot de passe

Attention : Cette action est irréversible. Toutes vos données seront supprimées définitivement.

Support Technique

Configuration système requise

Configuration minimale :

  • Navigateur : Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  • Connexion Internet : 5 Mbps minimum
  • RAM : 4 Go minimum
  • Processeur : Dual-core 2.0 GHz

Configuration recommandée :

  • Connexion Internet : 10 Mbps ou plus
  • RAM : 8 Go ou plus
  • Processeur : Quad-core 2.5 GHz ou plus
  • Carte graphique dédiée pour l'univers 3D

L'univers 3D ne se charge pas, que faire ?

Essayez ces solutions :

  1. Vérifiez que WebGL est activé dans votre navigateur
  2. Mettez à jour vos pilotes graphiques
  3. Désactivez les extensions bloquant le contenu
  4. Videz le cache de votre navigateur
  5. Essayez un autre navigateur
  6. Réduisez la qualité graphique dans les paramètres

Si le problème persiste, contactez le support avec votre configuration système.

Aurora est-il disponible sur mobile ?

Oui ! Aurora est accessible sur mobile via :

  • Site web responsive : Accessible depuis n'importe quel navigateur mobile
  • Applications mobiles : iOS 14+ et Android 10+ (en développement)

Note : Certaines fonctionnalités avancées (univers 3D) nécessitent un ordinateur.
