Introduction

Bienvenue sur Aurora. Nous nous engageons à protéger votre vie privée et vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons, partageons et protégeons vos informations lorsque vous utilisez notre plateforme éducative.

En utilisant Aurora, vous acceptez les pratiques décrites dans cette politique.

Collecte de Données

Nous collectons différents types d'informations pour vous offrir la meilleure expérience possible :

Informations que vous nous fournissez

  • Informations de compte : nom, prénom, adresse e-mail, date de naissance
  • Informations de profil : photo de profil, biographie, préférences d'apprentissage
  • Contenu généré : projets, devoirs, commentaires, messages

Partage des Données

Nous ne vendons jamais vos données personnelles. Nous pouvons partager vos informations uniquement dans les cas suivants :

Avec votre consentement

Lorsque vous autorisez explicitement le partage (ex: profil public, partage de projets)

  • Prestataires de services : hébergement, analyse, support client
  • Partenaires éducatifs : établissements scolaires (avec votre accord)
  • Obligations légales : autorités compétentes si requis par la loi
  • Protection des droits : pour faire valoir nos droits légaux
